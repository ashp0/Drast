# Changelog

## 0.2.0

- Added rich Drast symbol indexing for functions, variables, parameters, structs, fields, impl methods, protocols, enums, enum variants, runtime helpers, and imported C++ header symbols.
- Added hover cards with signatures, return types, source locations, and leading documentation comments.
- Added context-aware completions for locals, members, enum variants, C++ imports, standard runtime helpers, and snippets.
- Added document and range formatting.
- Expanded diagnostics for removed `fn` syntax, malformed imports, unsupported word operators, unclosed delimiters, and block-shape issues.
- Rebuilt the TextMate grammar and marketplace presentation for a polished Drast editing experience.

## 0.1.0

- Initial Drast VS Code extension.
- Added TextMate grammar and language configuration.
- Added fallback keyword/snippet completions.
- Added TypeScript LSP server with completions, hover, definition, workspace symbol scanning, and basic diagnostics.
